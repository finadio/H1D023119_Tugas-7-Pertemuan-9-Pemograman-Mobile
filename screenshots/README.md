## Cara Mengambil Screenshot

### Windows:
1. Buka aplikasi di browser/emulator
2. Tekan `Win + Shift + S`
3. Pilih area yang ingin di-screenshot
4. Paste di Paint atau langsung save

### Mac:
1. Buka aplikasi di browser/emulator
2. Tekan `Cmd + Shift + 4`
3. Drag untuk select area
4. File otomatis tersimpan di Desktop

### Chrome DevTools (untuk web):
1. Klik kanan ‚Üí Inspect
2. Toggle device toolbar (Ctrl+Shift+M)
3. Pilih device (iPhone/Pixel)
4. Klik menu 3 titik ‚Üí Capture screenshot
